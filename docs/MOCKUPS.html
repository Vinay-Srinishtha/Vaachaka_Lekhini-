Vaachaka Lekhini — Proposed Design Mockups

Same warm saffron / cream / teal palette as the Figma. Re-typeset in Lexend (UI) + Tiro Devanagari Hindi (mantras) + Tiro Telugu + Tiro Kannada (regional scripts). Responsive — phone and tablet from day one. Subtle motion at the moments that matter.

If this direction works, we lock it in and start translating these screens into Flutter widgets.

What this gallery represents

This is a proposed visual direction — same warm saffron / cream / teal palette as the Figma, but re-typeset with multi-script fonts (Lexend + Tiro Devanagari Hindi / Telugu / Kannada), tightened spacing rhythm, refined CTA hierarchy, and subtle motion where it earns its place (pulsing mic during voice training, ring breathing on the counter, animated waveform, smooth progress bars). The page itself is responsive — resize the window and the phones reflow from 1 column up to 4.

What's worth your review:

  • Are the colors right, or should the saffron be more muted / more vibrant?
  • Tiro Devanagari Hindi for mantra display — too ornate, just right, or do you prefer the sans-serif Noto?
  • Does the counter screen feel calm enough? (It's the screen users see most.)
  • Anything missing from the flows or screens?

Once you sign off, we lock the design system (tokens, fonts, components) and start building it as a Flutter design package — every screen above becomes a real responsive widget tree, identical look on phone and tablet.
